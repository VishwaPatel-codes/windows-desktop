# Windows Desktop

A cute fake desktop website inspired by old Windows and pixel-style games.

Live Demo: https://vishwapatel-codes.github.io/windows-desktop/

Features:
- Draggable desktop windows
- Notes app
- Pixel paint app with colors and tools
- Gallery folders
- Clock and taskbar
- Pixel grid background
- Cute retro UI

Built using HTML, CSS and a little JavaScript to practice layouts, styling and simple interactions.
